Step 3

Here I split the main section into a navigation menu and an about me. I try to plan the navigation menu early as I know I will need it. I write the text on an actual word document for spell check then copy/paste. I still don’t know what my website will look like so I’m going to go on Google and look for ideas.

Step 5

I got rid of the light gray background color so I could see how I liked it in white. Now that I have an example to work with I can get to work on replicating it. We don’t want it to be the exact same but the general layout is easy. It has a very small header, a large picture, a short about me, and links at the bottom. So all I will do now is rearrange my sections close to where they need to be.

Step 7

For the final touch-ups I simply remove the borders, add a background to the body and html, then begin spacing my elements where I want them. In an effort to make the site look similar to Franklina’s I crunched the text in the about me and sized/centered it with the image. This is just a rough draft and will likely go through changes as I make the rest of the site using the same template.
